## 🧩 Data Types
On Day 2, I learned about several fundamental data types in Python.

### 🔤 Strings
Strings are used to store text data. They are written inside single or double quotes.

### 🔢 Integers
Integers are whole numbers without decimal points.

### 🧮 Large Integers
Large integers can use underscores (`_`) to improve readability.  
This does not affect the actual value of the number.

### 📏 Float
Floats are numbers that contain decimal points and are used for more precise values.

### ✅ Boolean
Boolean values represent logical states and can only be `True` or `False`.

Understanding these data types is important because they affect how data is stored and how operations are performed in Python.

---

## ➗ Calculations in Python

Python supports basic mathematical operations and follows a specific order when evaluating expressions.

### 🧠 Basic Arithmetic Operations
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔺 Exponentiation

### 📐 Operator Precedence (PEMDAS)
Python follows the PEMDAS rule to determine the order of calculations:

1. 🧷 Parentheses `()`
2. ⚡ Exponents `**`
3. ✖️ Multiplication and ➗ Division
4. ➕ Addition and ➖ Subtraction

Following this order ensures that calculations are evaluated correctly and consistently.
